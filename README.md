# E-Commerce SQL Analysis

## Project Overview

This project analyzes user behavior on an e-commerce platform using SQL.

The objective is to understand:

- Funnel conversion rates
- Revenue performance
- Marketing channel effectiveness
- Product performance
- Customer journey insights

## Dataset

The dataset contains user interaction events:

- page_view
- add_to_cart
- checkout_start
- payment_info
- purchase

## SQL Concepts Used

- CTEs
- CASE Statements
- Aggregate Functions
- Conversion Funnel Analysis
- Revenue Analysis

## Key Insights

### Funnel Analysis

- 5000 users viewed products
- 1553 users added products to cart
- 826 completed purchases

### Marketing Performance

- Email had the highest conversion rate
- Organic search generated highest revenue
- Social media traffic had lowest conversion

### Product Performance

- Product 205 generated the highest revenue
